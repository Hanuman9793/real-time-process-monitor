# ⚡ Real-Time Process Monitor

🚀 A real-time system monitoring dashboard that provides live insights into CPU, memory, disk, and running processes.


---

## 📌 Overview

Most system monitors either overload users with data or hide what actually matters.
This project is built to provide **clear, real-time insights** into system performance in a simple and interactive way.

---

## ✨ Features

* 📊 Live CPU, Memory, Disk, and Network monitoring
* ⚙️ Real-time process tracking
* 🧠 Per-core CPU usage breakdown
* 📈 Interactive dashboard UI
* 🔄 Auto-refreshing data

---

## 🛠️ Tech Stack

**Frontend:**

* HTML

* JavaScript

**Backend (if used):**

* OS-level metrics (e.g., `os-utils`, `systeminformation` or similar)
